# AWS Security Reference Architecture

## Core Components
- IAM  
- GuardDuty  
- Security Hub  
- KMS  
- CloudTrail  
- VPC Flow Logs  

## Architecture Principles
- Least privilege IAM  
- Multi-account strategy  
- Private subnets  
- Service Control Policies (SCPs)  

## Controls
- CloudTrail mandatory  
- GuardDuty enabled  
- Encryption everywhere  
- IAM Access Analyzer
