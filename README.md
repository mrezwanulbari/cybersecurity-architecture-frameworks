# Cybersecurity Architecture Frameworks

This repository provides enterprise-grade cybersecurity architecture frameworks designed for financial institutions, critical infrastructure, and cloud-first organizations. It reflects real-world architectural patterns used in identity modernization, Zero Trust, cloud security, network segmentation, and regulatory compliance.

The goal is to provide **repeatable, scalable, and defensible architecture blueprints** aligned with industry standards and regulatory requirements.

---

## 📂 Repository Structure

### Enterprise Architecture (`enterprise-architecture/`)
Core architectural patterns for identity, network, data, and monitoring.

### Cloud Architecture (`cloud-architecture/`)
Azure and AWS reference architectures, cloud threat models, and workload protection patterns.

### Zero Trust (`zero-trust/`)
Identity-centric Zero Trust models, segmentation patterns, and enforcement strategies.

### Governance & Compliance (`governance-compliance/`)
NYCRR Part 500, ISO 27001, PCI-DSS control mappings and architecture alignment.

### Diagrams (`diagrams/`)
Architecture diagrams for enterprise, cloud, and Zero Trust.

### Documentation (`docs/`)
Overview, methodology, implementation guide, and roadmap.
